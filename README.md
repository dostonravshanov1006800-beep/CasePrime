# CasePrime 🎰🎁

**Интерактивный NFT казино на Telegram** - приложение для игры в слот-машины и открытие кейсов с NFT

## 📋 Описание

CasePrime - это FastAPI приложение, которое предоставляет игровой функционал:
- 🍒 **Slot Game** - классическая слот-машина с символами
- 🎁 **Case Game** - открытие кейсов с разными редкостями NFT
- 🏆 **Leaderboard** - таблица лидеров, обновляемая ежедневно
- 💰 **User Balance** - система учета баланса пользователей

## 🚀 Установка и запуск

### Требования
- Python 3.8+
- pip

### Шаги установки

```bash
# 1. Клонировать репозиторий
git clone https://github.com/dostonravshanov1006800-beep/CasePrime.git
cd CasePrime

# 2. Создать виртуальное окружение (опционально, но рекомендуется)
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить приложение
uvicorn main:app --reload
```

Приложение будет доступно по адресу: **http://localhost:8000**

## 📚 API Документация

После запуска приложения, откройте интерактивную документацию:
- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

### Доступные endpoints

#### 1. Получить баланс пользователя
```
GET /balance/{user_id}
```
**Ответ:**
```json
{
  "balance": 1000,
  "wins": 0
}
```

#### 2. Сыграть в слот-машину
```
POST /slot/{user_id}
```
**Ответ:**
```json
{
  "result": ["🍒", "💎", "7"],
  "reward": -50,
  "balance": 950
}
```

#### 3. Открыть кейс с NFT
```
POST /case/{user_id}
```
**Ответ:**
```json
{
  "item": "Rare NFT",
  "reward": 150,
  "balance": 1150
}
```

#### 4. Получить топ-10 лидеров
```
GET /leaderboard
```
**Ответ:**
```json
[
  {"user": 123, "score": 1000},
  {"user": 456, "score": 500}
]
```

## 🎮 Механика игр

### Slot Game 🍒
- **Символы**: 🍒, 💎, 7, ⭐
- **Выигрыш**: все три символа одинаковые → +500 коинов
- **Проигрыш**: иначе → -50 коинов

### Case Game 🎁
- **Common NFT** (70%) → +50 коинов
- **Rare NFT** (22%) → +150 коинов
- **Epic NFT** (7%) → +400 коинов
- **Legendary NFT** (1%) → +1000 коинов

### Leaderboard 🏆
- Обновляется каждые 24 часа
- Показывает топ-10 игроков по выигрышам
- Автоматически очищается при суточном сбросе
